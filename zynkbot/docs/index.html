Quick Start

Download and run — no compilation required

# Linux
sudo dpkg -i Zynkbot_0.9.0_amd64.deb

# Windows
# Run Zynkbot_0.9.0_x64-setup.exe as Administrator

Download from GitHub Releases →

Local models run CPU-only — can have 60+ second responses on some hardware. API models work at full speed. Full instructions → README.md
